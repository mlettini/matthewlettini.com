Work

    Simple, flexible, and visual time planning for teams.

  2. Timesheet Reminders

    Prompts to help customers build a habit with time tracking.

  3. Welcome Flow

    Greeting new trials and setting them up for success.

  4. Cancellation Flow

    Providing a pleasant and useful close account experience.

  5. Harvest for Mac v1

    Hotkey time tracking right from the desktop.

  6. Harvest for iPhone v2

    Simple on-the-go time and expense tracking.

    Multiple redesigns of Harvest’s marketing website.

    Javascript plugin to make long select boxes more user-friendly.
